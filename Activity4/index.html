Non-semantic Elements have no clear depiction of its content, it is typically an element with a generic tag name that can contain anything.

Class & ID Attribute are used to identify element, can be applied to any elements, used to label non-semantic elements, and styling or modifyng elements.

ID are used to identify a single element in HTML Document, can be used to cahnge its design or modify and retrive the element's values

CLASS are used to identify multiple elements

Semantic Elements have clear decription on what it contains, to both the developer and the browswer. have specific tag names. It does not change any styling on the actual document. It is used to structure the website in a cleaner way.
